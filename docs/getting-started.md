## 快速启动

准备一个干净的 Linux 服务器 (64 位，>= 4c8g)

```sh
curl -sSL https://github.com/open-c3/open-c3/releases/download/v2.6.1-latest/quick_start.sh | bash
```

通过您的浏览器访问 Open-C3，地址为 `http://your-openc3-ip/`

- 用户名: `open-c3`
- 密码: `changeme`
